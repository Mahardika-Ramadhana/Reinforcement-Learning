# Reinforcement Learning Projects

Learning Reinforcement Learning by building things.

## Projects

### 1. Snake AI (Deep Q-Learning)
An autonomous agent that learns to play Snake using a Vectorized Deep Q-Network (DQN).
- **Status:** Completed
- **Location:** `./snake`
- **Tech:** PyTorch, Pygame, NumPy

---

## How to Run

1. Navigate to the folder: `cd snake`
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python3 agent.py`
